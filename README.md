# devops

This workspace contains a small static website demo with three pages and one shared stylesheet.

Files created:
- index.html (Home)
- about.html (About)
- contact.html (Contact)
- css/styles.css (shared stylesheet)

Local preview:

Run a lightweight web server and open http://localhost:8000

```bash
python3 -m http.server 8000
# or, with Python 2
# python -m SimpleHTTPServer 8000
```
